# Channel Estimation Using MMSE in VLC System

## 📌 Project Overview

This project focuses on implementing **Minimum Mean Square Error (MMSE) channel estimation** in **Visible Light Communication (VLC)** systems to improve the accuracy of Channel State Information (CSI) and enhance communication reliability. The proposed method reduces estimation errors, mitigates noise and multipath interference, and outperforms conventional Least Squares (LS) estimation techniques.

## 🎯 Objectives

* Implement MMSE-based channel estimation for VLC systems.
* Improve signal quality and reduce Bit Error Rate (BER).
* Analyze the performance of MMSE under different Signal-to-Noise Ratio (SNR) conditions.
* Compare MMSE and LS channel estimation techniques.

## 🛠️ Technologies Used

* MATLAB
* Signal Processing
* Visible Light Communication (VLC)

* MMSE and LS Channel Estimation Techniques

## ✨ Features

* Accurate Channel State Information (CSI) estimation.
* Noise and interference mitigation.
* Improved reliability in low-SNR environments.
* Performance evaluation using MSE and BER metrics.

## 📊 Performance Metrics

* Mean Square Error (MSE)
* Bit Error Rate (BER)
* Signal-to-Noise Ratio (SNR)

## 📈 Results

Simulation results demonstrate that the MMSE estimator significantly reduces MSE and BER compared to the Least Squares (LS) method, particularly in low-SNR conditions.
